
### **Tipos de embudos de marketing:**

1. **Embudos de captación de leads**  
    Buscan convertir visitantes en clientes potenciales (leads) obteniendo sus datos a cambio de un beneficio (ej: descuento o contenido gratuito).
    
    - **Etapas:**
        
        - **TOFU:** Impacto inicial (anuncios, blog, etc.)
            
        - **MOFU:** Pop-up o banner con oferta a cambio de datos.
            
        - **BOFU:** Usuario deja su información.
            
2. **Embudos de cultivo de leads**  
    Permiten identificar qué leads están realmente interesados.
    
    - Se envían correos con llamados a la acción (ej: “Conocé nuestra nueva colección”) y se mide la interacción (aperturas, clics).
        
    - Si el usuario interactúa, demuestra interés y se lo considera más calificado.
        
3. **Embudos de ventas online**  
    Se dirigen a leads ya calificados para lograr una venta.
    
    - Se segmentan por intereses o datos demográficos.
        
    - Se les envían ofertas personalizadas (ej: “30% off en libros para jóvenes”).
        
    - Si no compran, se puede aplicar **remarketing** para intentar cerrar la venta.
        
4. **Embudos de up-selling**  
    Buscan que un cliente ya fidelizado realice una nueva compra con productos o servicios complementarios.
    
    - **Vía landing page:** Se muestra una oferta especial justo después de una compra.
        
    - **Vía email:** Días después de comprar, se envía una propuesta irresistible (oferta por tiempo limitado).
